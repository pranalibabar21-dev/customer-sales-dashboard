# Customer Sales and Churn Analysis Dashboard


This project is developed using Python, Machine Learning, and Streamlit for analyzing customer sales data, visualizing business insights, predicting sales, and analyzing customer churn.

The project is created as a Major Project for MCA Final Year under Savitribai Phule Pune University (SPPU).

---

# Project Features

- Customer Sales Analysis
- Product-wise Sales Visualization
- Region-wise Sales Analysis
- Monthly Sales Trend Analysis
- Customer Churn Analysis
- Machine Learning Sales Prediction
- Interactive Streamlit Dashboard
- Data Visualization using Charts and Graphs

# Machine Learning Model

## Algorithm Used

Linear Regression

Formula:

y = mx + b

The model predicts future sales based on Quantity and Price.


# Model Accuracy

- R² Score ≈ 86%
- Mean Absolute Error (MAE)


# Installation Steps

## Step 1: Clone Repository

```bash
git clone https://github.com/your-username/customer-sales-dashboard.git
```


## Step 2: Open Project Folder

```bash
cd customer-sales-dashboard
```

---

## Step 3: Install Required Libraries

```bash
pip install -r requirements.txt
```

---

# Run the Streamlit Dashboard

```bash
streamlit run app.py
```

```bash
python customer_analysis.py
